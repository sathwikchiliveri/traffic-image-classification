# Traffic Image Classification

A deep learning project for classifying traffic density from road images using Convolutional Neural Networks (CNNs) and transfer learning techniques.

## Overview

This project focuses on traffic density classification using computer vision and deep learning. The objective is to classify road traffic images into predefined traffic density categories, enabling intelligent traffic monitoring and smart transportation applications.

## Features

- Image classification using CNNs
- Transfer Learning (ResNet18)
- Data preprocessing and augmentation
- Model evaluation and visualization
- Prediction on unseen images

## Dataset

Singapore Traffic Density Dataset

Classes:
- Empty
- Low
- Medium
- High
- Traffic Jam
